1. Чтобы посмотреть содержимое директорий /etc, /proc, /home используется команда «ls». 

Например: 

ls /etc/
ls /proc/
ls /home/ 

Для просмотра содержимого файлов можно использовать несколько команд: 

a) cat - предназначена для отображения содержимого текстовых файлов.
b) nl - аналогична команде cat, но выводит номера строк в столбце слева.
c) less - выводит содержимое файла, но отображает его только в рамках текущего окна в режиме просмотра.
d) more - очень похожа на команду less, но есть некоторые отличия.
e) head - выводит на экран только первые 10 строк файла.
f) tail - выводит на экран последние 10 строк файла.

2. Команда «cat» предназначена для отображения содержимого текстовых файлов, копирования тестовых файлов в новый документ, присоединения содержимого текстового файла в конце другого текстового файла для их объединения.

a) Для создания двух файлов с данными использовал команду cat > file1, появилась пустая строка. Написал в ней «Some text of file1». И команду cat > file2 и дальше аналогично с первой командой. Чтобы объединить file1 и file2 использовал команду «cat file1 file2 > file3».

b) Для просмотра содержимого созданного файла использовал команду «cat file3»

c) Чтобы скопировать файл, дав ему новое название, использовал команду «mv file3 file4»
